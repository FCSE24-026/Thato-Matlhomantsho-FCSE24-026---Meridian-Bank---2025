# Amatic SC Font

This directory should contain the Google Fonts "Amatic SC" font files.

## How to Add the Font

1. Download Amatic SC from Google Fonts:
   - Visit: https://fonts.google.com/download?family=Amatic%20SC
   - Or go to: https://fonts.google.com and search for "Amatic SC"

2. Extract the downloaded ZIP file

3. Copy these files to this directory:
   - `AmaticSC-Regular.ttf`
   - `AmaticSC-Bold.ttf`

## Font Usage

The application will automatically load these fonts if present. If not found, it will use the system default font.

The fonts are used for the Meridian Bank branding and UI elements.
